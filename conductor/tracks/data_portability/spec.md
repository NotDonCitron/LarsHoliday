# Track: Data Portability (PDF Export)

## Goal
Enable users to export their search results and found deals into a clean, professional PDF report.

## Scope
- Create a new module `report_generator.py`.
- Use `FPDF2` or `ReportLab` (consistent with Tech Stack).
- Generate a PDF containing:
    - Search Summary (Dates, Guests, Filters).
    - List of Top Deals (Image placeholder, Title, Price, Rating, Link).
- Add an "Export PDF" button to the GUI.

## Tech Stack Compliance
- Library: `FPDF2` (preferred for simplicity) or `ReportLab`.

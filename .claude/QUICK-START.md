# Quick Start - Minimal Template

Dieses Template ist für **einfache Python-Scripts** gedacht.

## Wann nutzen?
- ✅ Einzelne .py Dateien oder wenige Dateien
- ✅ Schnelle Prototypen
- ✅ Automation Scripts
- ✅ Web Scraper
- ✅ CLI Tools
- ✅ Kleine Utilities

## Was ist enthalten?
- `CLAUDE.md` - Projektkontext für Claude
- Keine Hooks (kein Auto-Format, keine Checks)
- Keine komplexe Struktur
- Einfach und schnell

## Nächste Schritte
1. Fülle `CLAUDE.md` mit Infos über dein Projekt aus
2. Starte mit dem Coden!

Das war's. Kein Overhead, keine Komplexität.
